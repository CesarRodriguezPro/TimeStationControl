
this program is desineg to work with the API of timestation.com.

This is a very simple programs that helps to see problems with the hours of employees.
it creates a simple report and give information of possible wrong items.
for example:
    -employees clock in on Wrong locations or track if employees clock in in another location
    -employees forgoting to clock out on lunch 
    -employees forgoting to clock in after lunch

the interface is simple and has a very simple menu.
you will need to have pandas install and the API key that "www.Timestation.com" will give you.

please place the API Key in a text file and called it "Api_key.txt" and save it in the 
same forlder that this script
